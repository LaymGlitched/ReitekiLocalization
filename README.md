## Formatting

The following rules are applied to each localized text:

- `[b]` = Start Bold
- `[/b]` = End Bold
Example: "This is normal text, [b]This is bold text[/b]."

- `[i]` = Start Italic
- `[/i]` = End Italic
Example: "This is normal text, [i]This is italic text[/i]."
